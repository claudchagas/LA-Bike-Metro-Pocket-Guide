A comprehensive, print-ready 8-page pocket travel guide for exploring Los Angeles using bike and public transit as your primary transportation. Optimized for folding into a portable booklet.

## 🎯 Quick Start

**Option 1: Download PDF**
- Get `LA_Bike_Metro_Pocket_Guide.pdf` from [Releases](https://github.com/claudchagas/LA-Bike-Metro-Pocket-Guide/releases)
- Print double-sided on letter-size paper
- Fold in half → pocket booklet ✓

**Option 2: Compile from Source**
- Copy `source/main.tex`
- Paste into [Overleaf.com](https://overleaf.com)
- Click "Recompile" → Download PDF

See `COMPILATION_GUIDE.md` for detailed instructions.
